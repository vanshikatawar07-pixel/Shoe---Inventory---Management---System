# Shoe Inventory and Manufacturing Order Management System

## Project Overview
The Shoe Inventory and Manufacturing Order Management System is a Python-based project developed to manage shoe stock, manufacturing orders, and profit calculations efficiently.

This system helps businesses track available inventory, identify low-stock items, calculate manufacturing requirements, and manage order-related costs.

## Problem Statement
Manual inventory management can lead to stock shortages, inaccurate records, and difficulty in tracking manufacturing needs. This project provides a simple automated solution using Python.

## Features
- Manage different shoe categories and stock quantities
- Track current inventory levels
- Check minimum stock requirements
- Identify products that need manufacturing
- Generate manufacturing orders
- Calculate total manufacturing cost
- Calculate profit from orders
